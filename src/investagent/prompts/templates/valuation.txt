## 分析框架

### 1. 估值方法选择

在 `valuation_method` 中列出你使用的估值方法，常见方法包括：

- **正常化盈利收益率**：用正常化后的净利润 / 当前市值
- **自由现金流收益率**：Owner earnings（自由现金流）/ 当前市值
- **ROIC 再投资模型**：每股内在价值增长 = ROIC × 再投资率
- **分红折现模型（DDM）**：适用于稳定分红的成熟企业
- **资产重置法**：适用于重资产行业或清算价值评估

根据公司特征选择最合适的 1-3 种方法。

### 2. 穿透回报率计算

穿透回报（Look-through Return）= 正常化盈利收益率 + 每股内在价值增长率

- **正常化盈利收益率**：剔除一次性损益后的可持续盈利 / 市值
- **每股内在价值增长率**：基于 ROIC、留存收益比例、有机增长预期

输出三种情境下的 `expected_lookthrough_return`：

- **Bear（悲观）**：行业逆风、竞争加剧、增速下台阶
- **Base（基准）**：维持当前经营趋势
- **Bull（乐观）**：行业顺风、市场份额提升、效率改善

### 3. 摩擦调整回报率

从穿透回报中扣除摩擦成本，输出 `friction_adjusted_return`：

- **税负摩擦**：分红税、资本利得税（根据交易所和投资者身份）
  - A股：分红税 0-20%（持有期相关）
  - 港股：分红税 0-20%（取决于投资者类型）
  - 美股ADR：预扣税 10%
- **交易摩擦**：佣金、印花税、换汇成本
- **通胀摩擦**：实际购买力损耗

### 4. 是否达到门槛收益率

`meets_hurdle_rate`：Base 情境的摩擦调整后回报率是否 ≥ **{{ hurdle_rate_pct }}**（= 2× {{ currency }} 无风险利率 {{ risk_free_rate_pct }}）

- 如果 Base 情境摩擦调整后回报率 ≥ {{ hurdle_rate_pct }}，则为 true
- 否则为 false

### 5. 分析备注

在 `notes` 中记录关键假设和风险提示：

- 正常化盈利的调整依据
- ROIC 可持续性判断
- 再投资空间和资本配置质量
- 周期性因素对估值的影响
- 汇率风险（跨市场投资）
- 任何可能导致估值框架失效的因素

### 6. 内在价值与安全边际（芒格核心原则）

估算每股内在价值区间（`intrinsic_value_per_share`），并计算安全边际：

- 使用你选择的估值方法，推算 bear/base/bull 三种情境下的每股内在价值
- 内在价值的货币单位应与股价一致（如果报告货币与股价货币不同，需做汇率转换）
- **安全边际** = (base 内在价值 - 当前股价) / base 内在价值
- **价格判断**（必须严格按以下规则输出 `price_vs_value`，门槛收益率是硬性前提）：
  - EXPENSIVE：`meets_hurdle_rate` 为 false（不论安全边际多大，预期回报不达标即为贵）
  - CHEAP：`meets_hurdle_rate` 为 true **且** 安全边际 > 20%（达标且价格显著低于价值）
  - FAIR：`meets_hurdle_rate` 为 true **且** 安全边际 ≤ 20%（达标但安全边际不够充分）

芒格原则：伟大企业在 FAIR 价格即可考虑，普通企业必须 CHEAP。

## 输出要求

使用提供的工具输出以下字段：

- `valuation_method`：使用的估值方法列表
- `expected_lookthrough_return`：三种情境的穿透回报率（百分比，如 0.12 表示 12%）
- `friction_adjusted_return`：三种情境的摩擦调整后回报率（百分比）
- `meets_hurdle_rate`：Base 情境是否达到 {{ hurdle_rate_pct }} 门槛
- `intrinsic_value_per_share`：每股内在价值区间（bear/base/bull，注明 currency）
- `margin_of_safety_pct`：安全边际百分比（正数=低估，负数=高估）
- `price_vs_value`："CHEAP" / "FAIR" / "EXPENSIVE"
- `key_assumptions`：核心估值假设列表（正常化盈利依据、ROIC 可持续性、再投资空间等）
- `sensitivity_drivers`：敏感性驱动因素列表（哪些变量变化会显著影响估值结论）
- `notes`：其他分析备注
